FreeContent 3.0 Webdigest Tutorial


Using FreeContent to display your own html or php pages in Xoops. Freecontent features cache, search and Xoops native comments. The new wrapping method is XHTML compliant, but you can turn it off via the config.php file if you prefer the old method (set $fc_xhtml_compliance = 'no').

FreeContent 3.0 allows you not only to present easily a variety of contents but also to make concept/topic oriented WebIntelligence, to organize community discussion with always up to date and fresh content and information.

(For latest version of Freecontent go to: http://www.guanxicrm.com)

Freecontent's URL Rewriting function allows subsequent local urls (relatives) to be also showed in Xoops, this feature can be disabled by settings in config.php<>
Step1 upload your html or php content pages under the default content root dir (Xoops_root_dir/modules/freecontent/content).
your can change the default content root dir setting in config.php file (variable $fc_content_root_url) directory
Step2

go to Xoops admin panel > freecontent admin > List Contents

Provide appropriate path info for adding a new content connection.

give a Title -> this will be the name of menu item in the Freecontent Block

give a path-> this is where to find your html(php) file, for example, this tutorial file is in tutorial.html (relative to the default content root dir), you can also specify a full qualified url (ex. http://www.wjue.org) for Webdigest mode view.

give a detailed comment (optional but recommended, search will go here!) -> this is purely for your convenience.

Select the "Hidden?" option if you do not want the item to be showed in FreeContent's Menu Block (only for local file entries)
Select the "Digest?" option if you entered a full qualified url to webdigest.

Set offset to n to filter first n entries in your webdigest (most commercial pages' first links are ads)

You can set a regular expression to filter further unwanted contents.
Step3 Give access rights of FreeContent module and blocks to your users
Step4 Show the FreeContent Block (make FreeContent Block Visible), show MagicNews block if you wish.
Step5 Click on menu items in your Freecontent Block and enjoy your work !
